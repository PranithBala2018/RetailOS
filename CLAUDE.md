
# RetailOS - Claude Code Instructions

## Project Goal

Build a world-class Retail ERP + POS application capable of serving:

- Small shops
- Street vendors
- Tea shops
- Supermarkets
- Wholesale businesses
- Multi-branch retail chains

The application must support:

- Android
- Windows Desktop
- Web (future)
- Cloud
- Offline Mode

---

## Technology Stack

Frontend

Backend
- REST API

Database
- PostgreSQL

Authentication
- JWT

Architecture
- Clean Architecture
- Repository Pattern
- SOLID Principles

## Development Rules

Always:

- Write production-quality code.
- Never use placeholder code.
- Never skip error handling.
- Never leave TODO comments.
- Build reusable components.
- Write clean and scalable architecture.
- Add proper validation.
- Support offline mode.
- Support multi-business.
- Support multi-branch.
- Use responsive UI.
